# EDIDS

Adapter dell'interfaccia `Map` di J2SE 1.4.2 realizzato utilizzando la classe
`Hashtable` disponibile in CLDC 1.1.

## Struttura

- `myAdapter`: interfacce e implementazione dell'adapter.
- `myTest`: test JUnit e runner da linea di comando.
- `JUnit`: librerie necessarie all'esecuzione dei test.
- `doc/adapter`: documentazione Javadoc dell'adapter.
- `doc/test`: documentazione dei test.
